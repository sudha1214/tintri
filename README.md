## Introduction

Welcome to Tintri Coding Assignment.

## Installation
```bash
git clone https://github.com/sudha1214/tintri.git
cd tintri
npm install
```

##  Running the application
```
# This will build and launch the application on port 8000
npm start 
```
Navigate to http://localhost:8000/

## Running the tests
```
npm test
```
